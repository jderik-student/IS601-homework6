# Project Install Instructions

## install

1. clone
2. pip install -r requirements.txt

## testing

1. pytest
2. pytest --pylint
3. pytest --pylint --cov
4. pytest --num_records=100

## Start application
python main.py

## REPL Commands
1. add \<operand1\> \<operand2\>
2. subtract  \<operand1\> \<operand2\>
3. multiply  \<operand1\> \<operand2\>
4. divide  \<operand1\> \<operand2\>
5. getHistory
6. clearHistory
7. menu
8. exit
